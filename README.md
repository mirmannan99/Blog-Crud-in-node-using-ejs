# Blogs

It is a web app for writing and storing blogs. You may call it as a Blog Website.

Technologies used HTML(EJS),CSS,Javascript,runtime environment(Node.js),Monogodb database

1. your just need to run "npm i" in the terminal for the first time
2. this will install the required packages to run the webapp
3. to run the app you need to enter another command in the terminal " node app.js"
4. then go to the browser go to this url "http://localhost:3000/" your app will be working
5. then you can create read update and delete your blogs
6. the user interface is very easy to understand and very clean
